 📊 Financial Research Assistant using RAG

A Retrieval-Augmented Generation (RAG) application that enables intelligent analysis of corporate annual reports. The system processes financial reports from **Nestlé**, **Starbucks**, and **Coca-Cola**, allowing users to ask natural language questions and receive source-grounded answers generated by an LLM.

 🚀 Features

* 📄 Multi-document PDF ingestion using PyMuPDF
* ✂️ Semantic chunking for context-aware document segmentation
* 🔍 Vector search using ChromaDB
* 🧠 HuggingFace sentence embeddings for semantic retrieval
* 🤖 Qwen3 (Ollama) for local LLM-powered question answering
* 📚 Source-grounded responses with document citations
* 💬 Interactive Streamlit web application
* 📈 Cross-company financial report analysis

 🏗️ Tech Stack

* Python
* LangChain
* ChromaDB
* HuggingFace Embeddings
* Ollama (Qwen3:8B)
* Streamlit
* PyMuPDF

📂 Dataset

This project uses publicly available annual reports from:

* Nestlé
* Starbucks
* The Coca-Cola Company

💡 Example Questions

* What growth strategy did Starbucks mention?
* What are Nestlé's key business risks?
* How does Coca-Cola plan future growth?
* Compare the sustainability initiatives of Nestlé and Starbucks.
